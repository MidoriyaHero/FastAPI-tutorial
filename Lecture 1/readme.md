# Lecture 1
## Description:
This lecture will describe put get post delete method in FastAPI

When building APIs, you normally use these specific HTTP methods to perform a specific action.

Normally you use:

* POST: to create data.
* GET: to read data.
* PUT: to update data.
* DELETE: to delete data.
